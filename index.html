📝 How It Works:
  • Forex Pairs: Pip = 0.0001 (0.01 for JPY pairs), Standard lot = 100,000 units
  • Gold (XAUUSD): Pip = 0.01, Contract size = 100 oz
  • Cash Indices: Point = 1.0, Point values vary by instrument (e.g., US30: $1/point, STOXX50: ~$1.18/point)
  • Asian Indices: Point = 1.0, Point values optimized for risk management
  • European Indices: Point = 1.0, Contract size varies by broker
  • Cryptocurrencies: Point varies by price level, Contract size = 1 unit
  • Commodities: Point sizes vary by instrument type
  • Long Trade: Profit when close price > open price
  • Short Trade: Profit when close price < open price
